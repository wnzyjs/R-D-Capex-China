# China Chemicals R&D CAPEX Dashboard

This repository contains a static website for the China chemicals R&D CAPEX dashboard.

## Local entry point

Open `index.html`.

## GitHub Pages

This repo is configured for GitHub Pages via GitHub Actions.

To publish:

1. Create a new GitHub repository.
2. Upload the contents of this folder to the repository root.
3. Make sure the default branch is `main`.
4. In GitHub, go to `Settings` -> `Pages`.
5. Under `Build and deployment`, set `Source` to `GitHub Actions`.
6. Push to `main` or run the `Deploy static site to GitHub Pages` workflow manually.

After the workflow completes, the site will be available at your GitHub Pages URL.
